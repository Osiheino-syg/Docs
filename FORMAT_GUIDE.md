## What is React?

**React** is a JavaScript library for building user interfaces, particularly
single-page applications where you need a fast, interactive user experience.

> React uses a **component-based architecture** and **virtual DOM** to
> efficiently update and render UI components.

Developed by Facebook and maintained by Meta, React has become one of the most
popular frontend libraries in the world.

---

## Why React?

- **Component Reusability**: Build encapsulated components that manage their own state
- **Virtual DOM**: Efficient updates and rendering performance
- **Rich Ecosystem**: Large community, extensive libraries, and tools
- **Learn Once, Write Anywhere**: React Native enables mobile development

---

## Key Concepts

| **Concept**   | **Description**                           | **Example**         |
| ------------- | ----------------------------------------- | ------------------- |
| Component     | Reusable UI building block                | Button, Card, Form  |
| Props         | Data passed from parent to child          | `<Card title="Hi">` |
| State         | Component's internal data                 | `useState(0)`       |
| JSX           | JavaScript XML syntax                     | `<div>Hello</div>`  |
